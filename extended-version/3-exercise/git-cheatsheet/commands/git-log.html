GIT cheatsheet

git log

Show the history of commits done on the repository. This command is useful when you need to find an old commit or if you need to check where the branches are pointing to.

Options:

  • --all: shows the entire history of the repo
  • --graph: shows a grafical representation of the history
  • --decorate: add information on branches and HEAD pointer
  • --pretty=oneline: shows each commit on a single line
